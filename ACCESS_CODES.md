# Default Access Codes

## Super Admin Access Code
**Code:** `SUPERADMIN2024`

This is the default access code for Super Admin signup and login. It can be changed in the Super Admin portal under "Access Codes" section.

## Admin Access Code
**Code:** `ADMIN2024`

This is the default access code for Admin signup and login. Additional Admin access codes can be generated by Super Admin users in the Super Admin portal under "Access Codes" section.

---

**Note:** These codes are automatically initialized when the site loads for the first time. Make sure to change them after initial setup for security purposes.
